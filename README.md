# 🎨 ART – Virtual Reality Art Gallery

An immersive Virtual Reality (VR) art gallery built with **Unity** and the **XR Interaction Toolkit**. ART allows users to explore a virtual exhibition, interact with artworks using VR controllers, and experience a realistic gallery environment designed for an engaging museum-like experience.

> Developed as a collaborative Unity XR project for Meta Quest 3.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/prathamprabhu05/ART.git
```

2. Open the project using **Unity Hub**.

3. Install any missing packages from the Unity Package Manager.

4. Open the main scene.

5. Build or run the project on **Meta Quest 3**.
